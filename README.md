## Descrição do Projeto
Este projeto é um site educativo completo que aborda temas fundamentais para desenvolvedores e estudantes de tecnologia. O site serve como um guia de referência para conceitos importantes no desenvolvimento de software moderno, incluindo controle de versão, sistemas operacionais Linux, publicação de sites e princípios de segurança web.

O site foi desenvolvido com HTML5, CSS3 e JavaScript vanilla, seguindo boas práticas de desenvolvimento web responsivo e acessibilidade.

## Principais Pontos Revisados

### 1. Gerenciadores de Repositórios Remotos
- **Definição e importância**: Serviços que hospedam código fonte e facilitam trabalho colaborativo
- **Plataformas principais**: GitHub, GitLab, Bitbucket, Azure DevOps
- **Vantagens**: Backup automático, controle de versão distribuído, colaboração em equipe, CI/CD integrado

### 2. Comandos Linux Essenciais
- **Navegação de diretórios**: `ls`, `cd`, `pwd`, `mkdir`
- **Manipulação de arquivos**: `cp`, `mv`, `rm`, `cat`
- **Gerenciamento de permissões**: `chmod`, `chown`, `sudo`, `ps`

### 3. Git - Sistema de Controle de Versão
- **Comandos fundamentais**: `init`, `clone`, `add`, `commit`, `push`, `pull`
- **Trabalho com branches**: Criação, mudança, merge e resolução de conflitos
- **Fluxo de trabalho colaborativo**: Feature branches, pull requests e code review

### 4. GitHub Pages - Publicação de Sites
- **Configuração**: Criação de repositório específico (`usuario.github.io`)
- **Deploy automático**: Integração contínua com pushes para a branch main
- **Domínio personalizado**: Possibilidade de usar domínios customizados
- **HTTPS gratuito**: Certificado SSL incluído automaticamente

### 5. Segurança Web - HTTPS e DNS
- **HTTPS (HTTP Secure)**:
  - Criptografia SSL/TLS para proteção de dados
  - Autenticação de servidores
  - Integridade dos dados transmitidos
  - Prevenção de ataques man-in-the-middle

- **DNS (Domain Name System)**:
  - Tradução de nomes de domínio para IPs
  - DNSSEC para validação de respostas
  - DNS over HTTPS (DoH) para consultas criptografadas
  - Proteção contra spoofing e cache poisoning

- **Estilização**: CSS Grid, Flexbox, animações CSS
- **Ícones**: Font Awesome
- **Responsividade**: Design mobile-first
